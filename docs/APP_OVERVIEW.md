# Jetty Ferry Booking Mobile App - Overview

## Introduction
The Jetty Ferry Booking Mobile App is a comprehensive Flutter-based application designed to simplify ferry ticket booking and management for passengers traveling between various destinations.

## Key Features
- User authentication and profile management
- Real-time ferry schedule viewing
- Seamless ticket booking process
- Digital ticket management
- Route and destination information
- Payment integration
- Booking history tracking

## Target Audience
- Regular ferry commuters
- Tourists and occasional travelers
- Ferry service administrators
- Customer support teams

## Platform Support
- Android (Primary)
- iOS (Planned)
- Minimum Android SDK: 21 (Android 5.0)
